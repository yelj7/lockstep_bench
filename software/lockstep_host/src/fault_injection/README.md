# fault_injection

本目录对应 M08，负责测试模式故障注入配置、脚本调用、串口响应采集、注错记录和跳过状态。

边界规则：

- 故障注入仅属于测试模式入口。
- 未配置或跳过时记录为 `skipped`，不阻断 v1 `pass`。
- 成功、失败、超时都必须可追溯。
