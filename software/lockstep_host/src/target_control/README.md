# target_control

本目录合并 M05、M06、M07，负责目标连接与预检、自研片上调试访问、程序烧写、回读校验和运行控制。

内部边界：

- `connection`：目标连接、板卡 profile 业务校验、目标状态查询、预检结论和 `connection_ready` gate。
- `debug_access`：统一调试访问语义接口、后端适配、读写、复位、运行和中止。
- `programming`：镜像识别、写入范围解析、烧写记录、回读比较和运行控制记录。

边界规则：

- UI 和任务不保存 profile、模板或调试后端物理路径。
- 调试后端适配只停留在本目录内部，不泄漏到任务、报告或 UI 数据合同。
- 未连接或预检失败时不得烧写、回读、运行。
- 回读失败、回读不一致或缺少回读通过记录时不允许运行。
