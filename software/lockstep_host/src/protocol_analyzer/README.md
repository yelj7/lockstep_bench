# protocol_analyzer

本目录对应 M12，负责协议字段映射、协议事件、事务聚合、异常识别和波形时间定位。

边界规则：

- 协议规则缺失或处于占位状态时，不生成伪事务。
- 协议解析结果可以写入报告记录，但 v1 不作为 `pass` 强制证据。
- 波形渲染属于 `waveform_viewer/`，本模块只输出可定位的语义结果。
