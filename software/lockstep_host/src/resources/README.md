# resources

本目录对应 M03，负责安装包固化资源发现、校验、摘要生成和模式配置绑定。

边界规则：

- 读取 `resources/manifest.json` 并校验资源 ID、版本、相对路径和摘要。
- 校验板卡 profile、调试适配资源、报告模板和 lockstep trace profile 资源。
- lockstep trace profile 下统一保存板卡连接、通道映射、协议解析和显示配置等预置文件。
- 任务、报告和 UI 只保存逻辑 ID、版本和摘要，不保存资源物理路径。
