# waveform_viewer

本目录对应 M11，负责 VCD 加载、通道分组、统一时间轴、缩放、平移、局部窗口和协议事件定位显示。

边界规则：

- 只加载当前任务内的波形证据或明确传入的开发样例。
- 协议语义解析属于 `protocol_analyzer/`。
- 波形缺失或解析不可用时可记录为非阻断状态，v1 不阻断 `pass`。
