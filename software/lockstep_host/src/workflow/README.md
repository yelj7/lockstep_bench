# workflow

本目录对应 M04，负责研发/测试流程状态机、阶段前置条件、失败中止和 UI 操作可用性。

边界规则：

- 调度连接、烧写、回读、运行、采集、报告等模块，不直接执行硬件命令。
- 回读失败、回读不一致或缺少回读通过记录时关闭运行 gate。
- 未修复阻断错误不得继续依赖阶段。
