# acquisition

本目录合并 M09、M10，负责总线协议探测单元采集控制和采集帧流接收处理。

内部边界：

- `control`：采样窗口、触发方式、启用通道、采集标识和采集会话状态。
- `frames`：连续帧流接收、分帧、序号校验、错误帧、结束帧、原始采集文件和 VCD 生成。

边界规则：

- 配置不完整时不得启动采集。
- 异常帧必须标记失败或不完整。
- 结束帧必须匹配当前采集标识。
- 协议事务语义判断属于 `protocol_analyzer/`。
