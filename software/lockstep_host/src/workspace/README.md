# workspace

本目录对应 M02，负责项目工作区、研发/测试子工作区、任务生命周期、任务输入、任务索引和任务级产物路径。

边界规则：

- 研发模式只读写 `research_workspace/`。
- 测试模式只读写 `test_workspace/`。
- 日志、证据、波形和报告直接归属 `tasks/<task_id>/`。
- 任务输入和报告不得保存导入前外部路径。
