# reporting

本目录对应 M13，负责任务级产物索引、证据摘要、报告输入检查、`report.json` 生成和报告结论计算。

边界规则：

- 产物路径必须位于当前任务目录内。
- v1 只要求生成 `reports/report.json`。
- 有未修复阻断错误、缺少强制证据或回读失败时不得生成 `pass`。
- VCD、协议解析、采集文件、故障注入记录和 warning 可作为报告记录项，但不作为 v1 `pass` 强制证据。
