# lockstep_host

本目录是锁步研发测试系统上位机软件源码根目录，按软件设计说明书的源码库结构落地。

```text
lockstep_host/
  CMakeLists.txt
  deployment_config.json
  src/
  profiles/
  scripts/
  tests/
  examples/
  docs/
```

目录边界：

- `src/`：Qt/C++ 源码和模块库目标。
- `profiles/`：随安装包固化的板卡 profile 与调试适配配置来源。
- `scripts/`：开发期构建、运行、测试和打包脚本。
- `tests/`：自动化测试代码和夹具。
- `examples/`：离线演示和回归样例数据。
- `docs/`：上位机内部实现说明，补充项目级文档。
