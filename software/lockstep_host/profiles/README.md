# profiles

`profiles/` 保存安装包固化资源的源码来源。这些文件用于生成或复制到安装后的 `resources/` 目录，不保存用户任务数据。

- `board_profiles/`：板卡 profile 来源。
- `debug_service/`：自研片上调试服务适配配置来源。

任务、报告和 UI 只保存资源逻辑 ID、版本和摘要，不保存本机绝对路径。
