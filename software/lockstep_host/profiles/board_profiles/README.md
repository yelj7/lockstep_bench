# board_profiles

本目录保存板卡 profile 源文件。profile 进入安装包后应位于 `resources/board_profiles/`，并通过 `resources/manifest.json` 登记 ID、版本、相对路径和摘要。

profile 可描述固化资源内部路径，但任务、报告和 UI 不保存或展示这些物理路径。
