# 银河麒麟离线安装说明

## 支持平台

- 银河麒麟桌面操作系统 V10 SP1 General Release。
- 海光或兼容的 x86_64/amd64 处理器。
- 安装需要 root 或 sudo 权限，日常运行使用普通桌面用户。

## 构建离线套件

在干净的目标基线构建机安装 GCC/G++、CMake、Ninja 或 GNU Make（二选一）、Qt 5.15.2 开发包、hidapi 0.14.x hidraw 开发包、patchelf、pax-utils（提供 `lddtree`）、binutils、pkg-config 和 dpkg-dev，然后执行：

```bash
cd software/lockstep_host
./scripts/build-kylin-offline-package.sh
```

输出位于 `dist/kylin-v10-sp1-amd64/`。构建脚本会执行 CTest、ELF 架构检查、动态依赖闭包检查、资源清单生成和 SHA-256 生成。禁止在其他发行版生成正式交付包。

## 离线安装与验证

将整个交付目录带到目标机，在目录内执行：

```bash
./install.sh
```

安装脚本不会联网或调用 apt。安装完成后重新登录，使 `plugdev` 和 `dialout` 组权限生效，然后执行：

```bash
./verify-install.sh
lockstep-host
```

首次硬件验收应覆盖 CMSIS-DAP 枚举、目标连接、烧写、回读、运行、停止，以及串口枚举、打开和收发。USB 标识默认为 `0D28:0204`。

## 升级与卸载

安装更高版本 DEB 时再次运行新套件的 `install.sh`。卸载执行：

```bash
./uninstall.sh
```

卸载会移除 `/opt/lockstep-host`、桌面入口和专用 udev 规则，不会删除用户选择的工作区、任务数据、证据或报告。

## 故障定位

- `libqxcb.so` 加载失败：运行 `verify-install.sh`，检查私有 Qt 库和系统 X11/XCB 基础组件。
- CMSIS-DAP 无权限：确认已重新登录，检查 `groups` 是否包含 `plugdev`，并重新插拔设备。
- 串口无权限：确认 `groups` 是否包含 `dialout`。
- 安装包损坏：在交付目录运行 `sha256sum -c SHA256SUMS`。
