# 第三方依赖与许可证

## 随安装包内置

| 依赖 | 固定版本 | 用途 | 许可证 |
|---|---:|---|---|
| Qt Core/Gui/Widgets/Network/SerialPort | 5.15.2 | GUI、IPC、网络与串口 | LGPL-3.0-only 或商业许可证 |
| Qt xcb 及必要插件 | 5.15.2 | 银河麒麟 X11 桌面集成 | LGPL-3.0-only 或商业许可证 |
| hidapi hidraw 后端 | 0.14.x | CMSIS-DAP HID 访问 | BSD-3-Clause 或 GPL-3.0-only |
| libstdc++、libgcc_s | 构建工具链版本 | C++ 运行时 | GPL，带 GCC Runtime Library Exception |
| lockstep_debug_service | 与应用同版本 | 自研 CMSIS-DAP/JTAG 调试服务 | 项目自有代码 |

正式交付必须保留 Qt、hidapi 和 GCC 运行库的原始许可证文本及版权声明，包括适用的 LGPL-3、GPL-2、GPL-3 和 GCC Runtime Library Exception 材料。Qt 使用动态链接；不得禁止用户按 LGPL 要求替换对应共享库。若使用商业 Qt，则改为随商业许可材料交付。

## 由银河麒麟提供

glibc、动态加载器、pthread、dl、rt、m、X11/XCB、fontconfig、freetype、Mesa/OpenGL、libudev、udev、systemd-logind、内核 hidraw/tty 驱动和 dpkg 由目标操作系统提供，不复制到应用私有目录。

运行 `scripts/build-kylin-offline-package.sh` 后，以 `runtime-dependencies.json` 为最终逐 ELF 依赖清单；其中包含架构、SONAME、来源版本、SHA-256、许可证类别及“内置/系统提供”分类。

## 不需要的外部程序

当前运行路径不调用 OpenOCD、GDB、Python、Java 或其他下载器。目标连接、烧写、回读和运行由随包的自研 `lockstep_debug_service` 完成。
