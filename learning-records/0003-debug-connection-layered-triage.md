# 调试连接应按层排障

DAP 未枚举时先自动重枚举两次，仍失败才提示重新插拔；DAP 已枚举但打不开时优先检查进程占用和权限。JTAG 无链或无 IDCODE 时先 TAP reset 并重新连接两次，只有 DAP 正常且仍无响应时才提示重启板卡。DMI 排障必须建立在 IDCODE 已确认的基础上，不能把不同层级的故障统一解释成“重新插拔”。
